# SlackMailSync

Slackでメールを受信し、検索時にSlackのやり取りとメールのやり取りを時系列で見ることができるようになります。

## 機能

### メールの受信と返信

デフォルトでDMにメールが届きます。

### フィルター機能
指定のチャンネルへ転送したりブロックができます。
